# Database-Design
## Project Overview
This repository contains the Database Design and implementation project for Casual Infirmary Patient and Medical Management system.Completed by Elizabeth Amuzu September 2024. The sysytem is designed to efficiantly manage hospital data, including patient information, doctor details, medical records, appointments and department data.
